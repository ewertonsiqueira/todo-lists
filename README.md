# TodoList

Um aplicativo simples e eficiente de lista de tarefas construído com Vue 3, Vite e Bootstrap.

## 📋 Funcionalidades

- ✅ Adicionar novas tarefas
- 🗑️ Deletar tarefas
- 🔍 Filtrar tarefas por texto
- 📊 Filtrar tarefas por status (pendentes/concluídas)
- 📈 Estatísticas das tarefas

## 🚀 Tecnologias

- Vue 3 (Composition API)
- Vite
- Bootstrap 5
- Sass

## 💻 Instalação

```sh
npm install
```

## 🔧 Desenvolvimento

```sh
npm run dev
```

## 📦 Build para Produção

```sh
npm run build
```

## 📋 Requisitos

- Node.js ^20.19.0 || >=22.12.0
